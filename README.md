# 🌙 Ramadan Quiz THR

Interactive Ramadan quiz website with animated UI and QR reward system.

Users answer 10 questions about Ramadan.
If the score is high enough, they will receive a **QR Code reward (THR)**.

---

## ✨ Features

* 🌙 Ramadan themed UI
* 🏮 Lantern and star animations
* 🌴 Palm tree background
* 🎵 Background Ramadan music
* ❓ 10 interactive quiz questions
* 🎯 Score system
* 🎁 QR Code reward when passing the quiz
* 📱 Mobile responsive layout

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/FhrlRsknaa/ramadan-quiz.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```

---

## 🎮 How It Works

1. User opens the Ramadan quiz landing page
2. Click **Mulai Kuis**
3. Answer 10 questions
4. Score is calculated
5. If score ≥ 8 → QR Code reward appears
6. If score < 8 → encouragement message appears
